# Ubuntu 自动化运维容器项目

这是一个基于Ubuntu 22.04的Docker容器项目，集成了多种自动化任务，包括股票监控、新闻链接页面保存、IPTV频道更新等功能。
!!!不是一个标准项目！！！

## 项目概述

该项目构建了一个多功能的自动化运维环境，主要包含以下功能模块：

1. **股票监控系统** - 实时监控股票数据，分析师评级，生成HTML报告
2. **新闻爬虫** - 自动抓取新闻数据并保存
3. **IPTV频道管理** - 自动下载和更新IPTV频道列表
4. **Web服务器** - 使用Nginx提供Web服务
5. **远程访问** - 通过SSH进行远程管理

## 架构与技术栈

- **基础镜像**: Ubuntu 22.04
- **编程语言**: Python 3
- **Web框架**: Flask
- **数据库**: 本地文件存储及SQLite
- **定时任务**: Cron
- **Web服务器**: Nginx
- **浏览器自动化**: Selenium + Chrome Driver
- **OCR识别**: PaddleOCR
- **容器编排**: Docker + Docker Compose

## 功能特性

### 股票监控系统
- 实时获取股票数据
- 分析师评级跟踪
- 雪球网数据爬取
- 东方财富网数据获取
- 自动生成HTML报告
- 支持雪球网cookies认证

### 新闻爬虫
- 自动抓取新闻数据
- 支持多种新闻源
- 定期执行更新任务

### IPTV管理
- 自动下载M3U格式的电视频道列表
- 支持多类别频道分类
- 生成频道列表索引

### 系统服务
- SSH远程访问 (端口 4422)
- Web服务 (端口 80)
- 定时任务调度
- 日志记录与管理

## 定时任务

系统预设了以下定时任务：
- 每天凌晨1点执行股票监控检查
- 每天晚上11点下载IPTV频道列表
- 每8小时执行一次新闻爬取
- 每天晚上11点执行东方财富分析师数据抓取

## 端口映射

- **80端口**: Web服务 (映射到主机4400端口)
- **5001端口**: 股票监控应用 (映射到主机4401端口)
- **4422端口**: SSH远程访问 (映射到主机4422端口)

## 数据持久化

- `/data` - 主要数据存储目录
- `/paddle` - PaddleOCR模型存储目录

## 部署方式

使用Docker Compose进行部署：

```bash
docker-compose up -d
```

## 项目结构

```
my_ubuntu/
├── Dockerfile          # Docker镜像构建文件
├── docker-compose.yml  # Docker Compose配置
├── nginx.conf          # Nginx配置文件
├── default             # Nginx默认站点配置
├── iptv/               # IPTV频道管理模块
├── news/               # 新闻爬虫模块
├── stock_monitor/      # 股票监控模块
├── run/                # 运行时目录
└── ...
```

## 安全说明

- SSH服务允许root登录，密码为`rootpassword`
- 为Ubuntu用户设置了默认密码`ubuntupass`
- 使用特权模式运行容器以支持系统级服务

## 维护与扩展

此项目设计为可扩展的自动化运维平台，可根据需求添加新的爬虫或监控任务。
